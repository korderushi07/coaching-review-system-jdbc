# Coaching Review System (Java + MySQL + JDBC)

This is a console-based Coaching Review System developed using Java and MySQL with JDBC connectivity.

## Features

- Add coaching reviews
- View all reviews
- Update review ratings
- Delete reviews
- Search reviews by coaching name

## Technologies Used

- Java
- MySQL
- JDBC
- IntelliJ IDEA

## Database Structure

Table: reviews

Columns:
review_id
coaching_name
faculty_name
rating
review_text
student_name

## Project Objective

This project demonstrates CRUD operations using JDBC and shows how Java applications interact with MySQL databases.
